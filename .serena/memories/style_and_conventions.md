# Code Style and Conventions

## General
- **Language**: TypeScript
- **Formatting**: Managed by Biome. Indent style is tabs. Quotes are double quotes.
- **Linting**: Biome recommended rules are enabled. Organizing imports is handled by Biome.

## Component Structure
- Uses Next.js App Router conventions.
- Components are likely functional components using hooks.
- UI components are separated into `components/` directory.

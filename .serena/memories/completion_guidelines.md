# Completion Guidelines

Before marking a task as completed, ensure the following:

1. **Linting & Formatting**: Run `pnpm check` to ensure code style and linting rules are met.
2. **Type Checking**: Ensure there are no TypeScript errors.
3. **Verification**: If possible, run `pnpm build` to verify that the project still builds.
4. **Documentation**: Update any relevant documentation if the changes are significant.
